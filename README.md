# Claude Skills

A collection of reusable skills for Claude Code agents.

## Available Skills

| Skill | Description |
|-------|-------------|
| [collab](collab/) | Inter-agent collaboration protocol for tmux sessions |

## Installation

Symlink or copy to your Claude Code skills directory:

```bash
ln -s /path/to/claude-skills ~/.claude/skills
```

Or reference individual skills from your project's CLAUDE.md.
